## 1. สำหรับผู้เข้าร่วมทั่วไป (Attendees)

### 1.1 การลงทะเบียนเข้างาน
หากคุณยังไม่มีบัญชีผู้ใช้ หรือต้องการลงทะเบียนเข้าร่วมงาน ให้ทำตามขั้นตอนต่อไปนี้:
1. เข้าสู่หน้าแรกของเว็บไซต์
2. กดปุ่ม **"ลงทะเบียนเข้างาน"** หรือคลิกเมนู **"ลงทะเบียน"** ที่แถบด้านบน
3. กรอกข้อมูลส่วนตัวเบื้องต้นให้ครบถ้วน:
   - ชื่อ-นามสกุล, สถาบัน/คลินิก, เบอร์โทรศัพท์
   - อีเมล (สำคัญมาก: ใช้สำหรับ Login และรับใบเสร็จ)
   - รหัสผ่าน (ต้องมีความยาวอย่างน้อย 8 ตัวอักษร, มีตัวพิมพ์ใหญ่, ตัวพิมพ์เล็ก และตัวเลข)
4. เลือกแพ็กเกจที่ต้องการเข้าร่วม (Early Bird / Regular / Online)
5. ทำการชำระเงินผ่านการสแกน QR Code และ **แนบสลิปโอนเงิน** (อัปโหลดรูปภาพ)
6. กดปุ่ม **"ลงทะเบียน"** ระบบจะทำการส่งข้อมูลและสร้างบัญชีให้คุณทันที

### 1.2 การเข้าสู่ระบบครั้งแรก
1. กดคลิกที่ปุ่ม **"เข้าสู่ระบบ"** (ไอคอนรูปคน) มุมขวาบนของหน้าเว็บ
2. กรอก **อีเมล** และ **รหัสผ่าน** ที่ใช้สมัครสมาชิก
3. **การยืนยันอีเมล (Email Verification):**
   - หากคุณเพิ่งสมัครสมาชิกใหม่ ระบบจะส่งลิงก์ยืนยันไปที่อีเมลของคุณ
   - ระบบจะแจ้งเตือนให้คุณเข้าไปเช็คอีเมล (รวมถึงในกล่อง จม.ขยะ/Spam) และกดลิงก์ที่ได้รับ
   - *หมายเหตุ: หากกดเข้าสู่ระบบแล้วยังไม่ยืนยันอีเมล ระบบจะไม่ให้เข้าใช้งาน หากอีเมลหาย คุณสามารถกดปุ่ม "รับลิงก์ยืนยันอีกครั้ง" ภายในหน้าต่างแจ้งเตือนได้*
4. เมื่อยืนยันอีเมลและเข้าสู่ระบบสำเร็จ ปุ่มเมนูจะเปลี่ยนเป็น **"My Dashboard"**

### 1.3 การลืมรหัสผ่าน
1. ไปที่หน้าเข้าสู่ระบบ และคลิกปุ่ม **"ลืมรหัสผ่าน?"** (ไอคอนกุญแจ) ข้างปุ่มเข้าสู่ระบบ
2. กรอกอีเมลของคุณ จากนั้นระบบจะส่งลิงก์สำหรับตั้งรหัสผ่านใหม่ไปยังอีเมลที่ระบุ

### 1.4 การใช้งานหน้า Dashboard
เมื่อเข้าสู่ระบบสำเร็จ ให้คลิก **"My Dashboard"** คุณจะพบกับเมนูจัดการข้อมูลส่วนตัว ดังนี้:
- **ข้อมูลการจอง (My Registrations):**
  - ติดตามสถานะการชำระเงิน (🟢 อนุมัติแล้ว / 🟡 รอตรวจสอบ / 🔴 ปฏิเสธ)
  - **ดาวน์โหลดใบเสร็จ / ใบประกาศ:** เมื่อสถานะเป็น "อนุมัติแล้ว" จะมีปุ่มโผล่ขึ้นมาให้คุณคลิกดาวน์โหลดเอกสารเป็นไฟล์ PDF ได้ทันที
- **ข้อมูลส่วนตัว (Profile):** แก้ไขชื่อ, เบอร์โทร, สถาบัน ได้ตลอดเวลา
- **ออกจากระบบ (Logout):** เพื่อความปลอดภัยในการใช้งานคอมพิวเตอร์สาธารณะ

## 2. สำหรับตัวแทนจำหน่าย/บูธ (Vendors)

### 2.1 การจองพื้นที่บูธ
1. กดเมนูเข้าสู่ระบบ และเลือกแท็บ **"จองบูธ (Vendor)"**
2. กรอกข้อมูลบริษัท, สินค้า, เบอร์โทรศัพท์
3. เลือกแพลนและขนาดพื้นที่ที่ต้องการ (มีราคาแจ้งในแต่ละแบบ)
4. ชำระเงินและแนบสลิป
5. สถานะของบูธจะไปแสดงใน **My Dashboard** แอดมินจะติดต่อกลับตามเบอร์ที่ให้ไว้หลังจากตรวจสอบสลิปเรียบร้อยแล้ว

## 3. สำหรับผู้ดูแลระบบ (Admin & Staff)

### 3.1 การเข้าสู่ระบบของ Admin
1. ไปที่หน้า เข้าสู่ระบบ บนหน้าเว็บไซต์ (หรือกดเมนูไอคอนคน)
2. กรอกข้อมูลสำหรับสิทธิ์ตามระดับ:
   - **Staff (มองเห็นข้อมูล ควบคุมอนุมัติได้ แต่ไม่เห็นรายได้หรือการลบข้อมูล):** Username: `staff` / Password: `staff123`
   - **Super Admin (เข้าถึงได้ทุกฟังก์ชัน):** Username: `admin` / Password: `admin123`
3. ระบบจะพาเข้าสู่หน้า `admin.html` อัตโนมัติ

### 3.2 การตรวจสอบและอนุมัติสลิปโอนเงิน
ภายใต้หน้า Dashboard ของ Admin:
- คุณจะเห็นตาราง **Recent Registrations** และ **Booth Bookings**
- **สลิปโอนเงิน (Slip):** หากคลิกที่รูปสลิป ระบบจะโชว์ภาพขยายให้เห็นเต็มจอ (แก้ไขปัญหาหน้าต่างซ้อนกันเรียบร้อยแล้ว)
- **เครื่องมือตัวช่วย:** สามารถกดปุ่มเครื่องหมายคัดลอก (Copy) ข้างๆ จำนวนเงิน และ วันเวลา เพื่อนำไปวางตรวจสอบในแอป ธนาคารบนมือถือได้ทันที
- **การอนุมัติ:** หากสลิปถูกต้อง กดปุ่ม **"Approve" (สีเขียว)** เพื่อยืนยัน หรือ **"Reject" (สีแดง)** เพื่อให้ผู้ใช้งานอัปโหลดใหม่

### 3.3 การ Export ข้อมูล
- ในหน้า Admin จะมีปุ่ม **"Export CSV"** ทั้งในตารางของผู้ร่วมงาน และของบูธแสดงสินค้า
- เมื่อกดแล้วระบบจะทำการดาวน์โหลดไฟล์ตาราง Excel/CSV ออกมาอัตโนมัติ

## 4. การแจ้งปัญหาและการช่วยเหลือ (Support)

- **AI Chatbot Assistants:** ที่มุมขวาล่างของหน้าจอทุกหน้า จะมีหุ่นยนต์ผู้ช่วย (Chatbot) คอยตอบคำถามเบื้องต้น (เช่น วันจัดงาน, การจอดรถ, สถานที่)
- **ติดต่อ Admin ทันที:** ในหน้าเข้าสู่ระบบ จะมีบรรทัดล่างสุดเขียนว่า **"มีปัญหาการเข้าสู่ระบบ? ติดต่อแอดมิน"** หากพบปัญหา สามารถกดเพื่อเปิดหน้าแชทติดต่อทีมงาน Support ของ DentX ได้โดยผ่านระบบ Chat ภายในเว็บครับ
